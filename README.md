# homebrew-fermata

Homebrew tap for [Fermata](https://fermata.run) — native macOS AI agent orchestrator for Claude Code.

## Install

```bash
brew tap kelios-io/fermata
brew install --cask fermata
```

Or one-shot:

```bash
brew install --cask kelios-io/fermata/fermata
```

Fermata updates itself via Sparkle once installed.
